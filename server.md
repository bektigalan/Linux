# Instalasi Web Server

## Instalasi Apache2
```
sudo apt get update
```
```
sudo apt get install apache2
```

## Instalasi Mysql
```
sudo apt get install mysql-server mysql-client
```

## Instalasi PHP
```
sudo apt get install php php-apache2 php-mysql phpmyadmin
```

# Tweak Web Server
```
a2enmod rewrite
```
